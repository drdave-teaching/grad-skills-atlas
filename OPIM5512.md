## 🔗 The shared core — skills that show up in *every* one of your grad classes
These are the highest-leverage skills to drill, because they transfer to 5509, 5641, and beyond:
- **① GitHub & reproducibility** — repos, PRs, `requirements.txt`, stable data links.
- **⑧⑩ The ML methodology & evaluation** — read→split→**scale-on-train**→fit→evaluate; metrics + plots.
- **③ Pipelines, cross-validation, tuning** — leakage-safe model selection.
- **② Explainability** — permutation importance / PDPs (model-agnostic, works on any model in any course).

> Everything else is course-specific *theory* (transformers, TSFresh, SMOTE). The shared core is the **spine**.
